# zepto-retail-analytics
Built an end-to-end data analytics project to analyze product pricing, discount strategies, and inventory availability for a quick-commerce retail dataset. Cleaned and explored data using Python (Pandas, Jupyter), performed SQL-based analysis in MySQL to derive category and stock-level insights, and designed interactive Power BI dashboards

## Tools Used
- Python (Pandas, Jupyter Notebook)
- MySQL Workbench
- Power BI

## Project Workflow
1. Data cleaning and exploratory analysis using Python
2. SQL-based analysis for pricing and inventory insights
3. Interactive dashboards built using Power BI

## Key Business Insights
- Identified high-discount products impacting margins
- Analyzed category-wise pricing patterns
- Detected low inventory and out-of-stock risks
- Compared MRP vs discounted selling price trends
